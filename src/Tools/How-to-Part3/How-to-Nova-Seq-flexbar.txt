How to do a Nova Seq run example 6

Flexbar runs on the Unrecognized files for a lane.


Example based on xR074 lane1 group8

Prerequisite:

- runs on Ubuntu OS
- need to install flexbar package



1. Preparing the flexbar tsv file and script

- get tsv file of barcodes and sample names

- create a directory to gather flexbar info

example xr074/lane1/flexbar

need files:

adapter.3.fa
deal.with.barcode.pl
demux.sh
script.sh


I copy the flexbar tsv file to one that is specific to the run

cp Flexbar_HistoneHuman_PAREseq_MurnJ_1125I-07.txt xR074-L1-G8-flexbar-barcodes.tsv


edit script.sh 

- to cover the correct read1 Unrecognized
- cover the tsv file

cat script.sh


echo $STY
echo START Flexbar Read1
date
sh demux.sh xR074-L1-G8-flexbar-barcodes.tsv Undetermined_S0_L001_R1_001.fastq.gz
date
echo END


2. Need the Unrecognized for the lane, comes after normal bcl2fastq script finishes


3. Make a directory to run for the group.

mkdir group8

put all the flexbar files there with the Unrecognized


adapter.3.fa					Flexbar_HistoneHuman_PAREseq_MurnJ_1125I-07.txt
deal.with.barcode.pl				script.sh
demux.sh					xR074-L1-G8-flexbar-barcodes.tsv


4. run script

./script



This completes Read 1


5. make script for getting Read 2 by capturing the sample tile locations from Read 1 




===============================================================


ls flexbarOut*.gz > list.txt



Create subseq lists



Need to do:


zcat flexbarOut_barcode_L-0-1.fastq.gz | grep 1:N | sed 's/@//' | cut -d ' ' -f1 > subseq_L-0-1. txt


Need:

	- List of fastq.gz
	- Output name


ls flexbarOut*.gz > list.txt delete unassigned



cut -d _ -f3- list.txt | sed 's/.fastq.gz//' > subseq-raw-name.txt


sed 's/^/subseq-/' subseq-raw-name.txt | sed 's/$/.txt/' > subseq-names.txt 






Make list of Unrec read 2 with liens to match subseq list

vi unrec-list.txt

Undetermined_S0_L001_R2_001.fastq.gz

Yank yy

Put (number of lines in list.txt minus 1) with p





$ sed 's/^/\/home\/sequser\/Software\/build\/seqtk\/seqtk\/seqtk subseq /' unrec-list.txt > tmp1.txt


$ paste -d ' ' tmp1.txt subseq-names.txt | sed 's/$/ | gzip  > /' > tmp2.txt




Setup READ2 names


$ sed 's/^/READ2-/' list.txt > tmp3.txt



$ paste -d '\0' tmp2.txt tmp3.txt > script-filter-read2-seqtk.sh



Need create subseq list first


$ sed 's/^/zcat /' list.txt | sed 's/$/ | grep 1:N | cut -d " " -f1 | more | cut -d "@" -f2 > /' > tmp4.txt


paste -d '\0' tmp4.txt subseq-names.txt  > script-zcat.sh



Do  script-zcat.sh 
then script-filter-read2-seqtk.sh
