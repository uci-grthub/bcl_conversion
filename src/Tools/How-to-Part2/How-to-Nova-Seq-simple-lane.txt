How to do a Nova Seq run example 1

Simple case of entire lane being dual index and paired.

Was run xR035.

attached is the working directory where I process the run. It includes

- notes on the run including the contents of the info from RunInfo.xml
- all scripts used for bcl2fastq and processing
- all scrips for collecting the information and packing the notifications

Steps done:

1. Get Sample Sheet and make the processing Sample Sheets

each lane directory (lane1 lane2 etc)  contains the information from the lane.

For instance, in lane1

1a-bc.txt is 1st index
1b-bc.txt is 2nd index

1-dual-bc.txt is the dual index, crated via
paste -d , 1a-bc.txt 1b-bc.txt > 1-dual-bc.txt

create the prefix file by knowning the number of lines, for lane 1 it is 5 lines.

Used website

https://pinetools.com/generate-list-numbers

to make the prefix (used to use a sed/awk script, but because you do a bunch of steps, using this GUI is faster)

prefix-1.txt is the output

1.txt is the body of the Sample Sheet crated via
paste -d '\0' prefix-1.txt 1-dual-bc.txt > 1.txt


Create the Sample Sheet by concatenating header for Sample Sheet with body via

cat ../../SampleSheet-header.csv 1.txt > SampleSheetxR035-lane1-dual-bc.csv

This can be done before the NovaSeq creates the run directory




2. Create the bcl2fastq script once the run directory is created and the RunInfo.xml file is created.

The key info is

- directory name

20250310_LH00626_0042_B22J2YCLT4

- read sizes

                <Reads>
                        <Read Number="1" NumCycles="151" IsIndexedRead="N" IsReverseComplement="N"/>
                        <Read Number="2" NumCycles="10" IsIndexedRead="Y" IsReverseComplement="N"/>
                        <Read Number="3" NumCycles="10" IsIndexedRead="Y" IsReverseComplement="Y"/>
                        <Read Number="4" NumCycles="151" IsIndexedRead="N" IsReverseComplement="N"/>
                </Reads>


file

new-command-bcl2fastq2-20-xR035-paired-dual-bc-lane1.sh

is the script created.


Note that masking in the excel file says 151,8,8,151 so teh maskings needs to be:

Y151,I8N2,I8N2,Y151 


3. Create the post processing script process-lane-1-xR035-paired-dual-bc.sh

echo $STY
date
echo ~/Facility/Tools/postprocess_hiseq_lane_dual_linux_os.pl SampleSheetxR035-lane1-dual-bc.csv 2 1 \"VilaE_WGS_Pool1\" 1
~/Facility/Tools/postprocess_hiseq_lane_dual_linux_os.pl SampleSheetxR035-lane1-dual-bc.csv 2 1 "VilaE_WGS_Pool1" 1
date

Need:

- Sample Sheet name SampleSheetxR035-lane1-dual-bc.csv

- paired vs single run 2

- lane number 1

- Name of the run inside the "" - note we do not all allow blank spaces in the name. 


Extra stuff

- I use GNU screen for persistent terminal connections so $STY is the name of the session

- I echo the command to remind me of what is being done.

- I use date to time the run


4. Process the run

Put Sample Sheet, bcl2fastq script and processing script in the run directory

wait till RTAComplete.txt and/or CopyComplete.txt

This is when the data is in

- It looks at the S number in the fastq files so 1 will look for S1
- It runs linearly through the Sample Sheet file and assumes the Unrecongized will come at the end. So I put the file in te last batch
